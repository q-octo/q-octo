# Q-Octo

## Setup

1. Clone this repo
2. Run `git submodule update --init`

VSCode with the PlatformIO extension or CLion with the PlatformIO Plugin is recommended.

### Web Dashboard

`curl -fsSL https://bun.sh/install | bash` (https://bun.sh/docs/installation#installing)
